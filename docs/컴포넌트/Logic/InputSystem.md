# InputSystem

---

## 설명

기존 input 기능 랩핑 컴포넌트

## 예시 코드

### 멤버 함수

### public

| 이름 | 내용 |
| --- | --- |
| IsKeyPressed | 키를 누르고 있는지 확인하는 함수 |
| IsKeyDown | 키를 눌렀는지 확인하는 함수 |
| IsKeyRelease | 키를 땠을 때 확인하는 함수 |

### 멤버 변수

### private

| 이름 | 내용 |
| --- | --- |
| input | 컴포넌트 생성 시 객체 생성, 키 입력을 처리할 클래스를 참조할 변수 |
